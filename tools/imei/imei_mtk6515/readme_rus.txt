Использование: imei.xxx <15digit_IMEI_1> [<15digit_IMEI_2>] [выходной файл]

Если оба IMEI вы задали в правильном формате, будет создан файл с именем
MP0B_001. Теперь вы можете переместить его в /nvram/md/NVRAM/NVD_IMEI/MP0B_001 
(путь для новsх прошивок) или в /data/nvram/md/NVRAM/NVD_IMEI/MP0B_001 (для 
старых).

Примечание 1: устройство должно быть rooted!
Примечание 2: имя выходного файла не должно состоять из 15 цифр, иначе оно
будет рассматриваться, как IMEI.

Изменения по сравнению с оригинальной версией.
1. Проверка IMEI на корректность контрольной суммы (последняя цифра).
   Спасибо за идею, f_mulder!
2. Добавлен параметр для задания имени файла.
3. Проверка существования выходного файла. Если существует, то выход с ошибкой.
4. Проверяем, разрешена ли запись в файл. Может быть запрещена из-за неверного
   указания пути или ограничения прав доступа.
5. Имя файла по умолчанию - "MP0B_001".
6. Не позволяем задавать 2 одинаковых номера IMEI. Это может привести к отказу
   в обслуживании некоторыми операторами GSM.
